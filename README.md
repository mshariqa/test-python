# Python Test Project

## Quick Start

1. Create virtual environment:
```bash
uv venv
```

2. Install dependencies:
```bash
uv pip install -r requirements.txt
```

3. Add your OpenAI API key in `.env`:
```
OPENAI_API_KEY=your_api_key_here
```

4. Run the code:
```bash
# Run AI agent
uv run ai_agent.py

# Run async examples
uv run async.py
```
